# 🔐 AI-Enabled Smart Surveillance System using IoT

## 📌 Project Description
This project is an AI-based smart surveillance system that performs real-time face detection and recognition using OpenCV and DeepFace.

---

## 🚀 Features
• Real-time face detection  
• Face recognition (Known vs Unknown)  
• Image capture of unknown person  
• Email alert system  
• Smart door lock system (Future enhancement)  
• Raspberry Pi-based deployment (Future scope)
